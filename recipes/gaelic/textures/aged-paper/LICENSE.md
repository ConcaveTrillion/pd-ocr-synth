# Bundled aged-paper textures

These PNGs are 100% procedurally generated by
`scripts/generate-paper-textures.py`. No scanned material, no
upstream IP, nothing third-party.

They are dedicated to the public domain under
[CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).
You may use, modify, and redistribute them with or without attribution.

To regenerate (and for the algorithm details), see the script:

```bash
uv run python scripts/generate-paper-textures.py
```

The same defaults produce byte-identical PNGs because the generator
seeds NumPy with fixed values per output index.
